Projects

YGO INDEX:

YGO INDEX is an informational and deck building website about the anime series known as Yu-Gi-Oh!. The website contains images and information about each sequel/generation of Yugioh!. In addition, there is a webpage where users can search up cards from the anime. This includes all types of cards such as Monsters, Spells, and Trap cards. When viewing the card, users can add it to their own deck in the "My Deck" section. If users wants to remove a specific card from their deck, they can by clicking on the "Remove" button. The technologies used to build this project are Node.JS, Javascript, Axios, Express, HTML & CSS.

Spit Time:

Split Time is an informational website where users can learn about exercises and create a workout schedule. Users can also edit and delete their workout splits. If a user is new to exercising and doesn't know any kind of exercise, they can check the exercise page where there is a list of exercises. Each exercise contains a link to a tutorial YouTube video. The technologies used to build this project are JavaScript, HTML, CSS, Java, Spring Boot, and PostgreSQL.
